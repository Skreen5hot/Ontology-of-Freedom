## HTML Demonstrations

### [pos-tagger-demo.html](pos-tagger-demo.html)
Part-of-speech tagging demonstration

Interactive moral rule builder

### [scenario-builder.html](scenario-builder.html)
Moral scenario construction tool
- Build test scenarios
- Export for testing

### [violation-detector.html](violation-detector.html)
Moral violation detection demonstration
- Input: Scenarios
- Output: Detected violations
